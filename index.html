MobileUI

Development of native App, multiplatform, in a simple way, using HTML, CSS and JavaScript

MobileUI es un framework libre, open source, ultraliviano, optimizado para Phonegap y especialmente para Phonegap Build, que permite desarrollar en forma rápida y sencilla App Profesionales para iOS, Android y Windows Phone, con interfaz de usuario auto-escala adaptable a los diversos tamaños, usando tecnologías conocidas como HTML, CSS y JavaScript.
